OS Builds

The os.radix system type builds RADix OS images. Crimson currently uses this for x86 GRUB VM images with either a framebuffer terminal profile or a RADCompositor window-manager profile.

Terminal Profile

radbuild build os --settings settings.terminal.json --json-events

Expected artifacts:

radixkernel-x86-64-grub-terminal.iso
radixkernel-x86-64-grub-terminal
radix-rootfs.ext4
radix-fat32.img
run-radix-vm.sh
SHA256SUMS

Window Manager Profile

radbuild build os --settings settings.wm.json --json-events

The window-manager profile enables RADCompositor and Slint shell chunks while keeping the same package and artifact flow.

RADix Menuconfig

radbuild menuconfig --settings settings.terminal.json --list
radbuild menuconfig --settings settings.terminal.json --apply-profile wm --json
radbuild menuconfig --settings settings.terminal.json --enable-package radlib-ui --resolve-packages --json

Package edits are dependency-aware. Enabling a package automatically enables its required packages from the configured .radpm index.

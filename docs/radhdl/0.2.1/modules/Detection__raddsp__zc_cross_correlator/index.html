zc_cross_correlator

Documentation version: 0.2.1 / Package: Detection / Source package: raddsp

Streaming Zadoff-Chu cross-correlator primitive. Computes correlation energy between an incoming complex stream and a stored reference sequence.

Use Cases

  • Reusable RadHDL building block for graph-generated FPGA systems.

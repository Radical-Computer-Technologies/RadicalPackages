radif_i2s_axis

Documentation version: 0.2.1 / Package: radif / Source package: radif

Bidirectional I2S to AXI-Stream bridge. Converts stereo I2S samples to AXI-Stream frames and AXI-Stream frames back to I2S while exposing clocking and enable controls through a RADIF register interface.

Register interfacesRegister maps: radif_i2s_axis (7 registers)Testbenches: 1

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Memory-mapped control/status integration with software-visible register maps.

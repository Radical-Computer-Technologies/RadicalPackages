raddsp_axis_radfft_radix2_tdp

Documentation version: 0.2.1 / Package: Transform / Source package: raddsp

Radix-2 true-dual-port-memory AXI-stream RadFFT engine. Uses dual-port scratch memories to schedule butterfly passes with predictable memory access patterns.

Register interfaces

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

radif_axi_lite_to_reg

Documentation version: 0.2.1 / Subpackage: radif_axi / Source package: radif

AXI4-Lite slave bridge to the RadIF register transaction interface. Turns AXI read/write handshakes into one-cycle register request pulses and returns ready/valid responses to the bus.

Register interfacesTestbenches: 1

Use Cases

  • Memory-mapped control/status integration with software-visible register maps.

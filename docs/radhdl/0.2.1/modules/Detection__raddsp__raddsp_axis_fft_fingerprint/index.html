raddsp_axis_fft_fingerprint

Documentation version: 0.2.1 / Subpackage: dsp_detection / Source package: raddsp

AXI-stream spectral fingerprint extraction core. Condenses FFT output into compact bin-energy features that can be compared against stored fingerprints.

Register interfacesRegister maps: raddsp_axis_fft_fingerprint_inferred (16 registers)Testbenches: 2

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

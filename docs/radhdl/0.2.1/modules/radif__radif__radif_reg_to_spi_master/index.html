radif_reg_to_spi_master

Documentation version: 0.2.1 / Package: radif / Source package: radif

Register-controlled SPI master. Provides a small software-visible register bank that launches single-word SPI transfers with programmable SCLK timing, chip-select polarity, clock polarity, and clock phase.

Register interfacesRegister maps: radif_reg_to_spi_master (6 registers)Testbenches: 1

Use Cases

  • Memory-mapped control/status integration with software-visible register maps.

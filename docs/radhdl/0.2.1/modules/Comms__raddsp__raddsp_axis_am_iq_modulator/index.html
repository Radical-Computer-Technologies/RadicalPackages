raddsp_axis_am_iq_modulator

Documentation version: 0.2.1 / Subpackage: dsp_comms / Source package: raddsp

AXI-stream AM IQ modulator. Multiplies a signed envelope stream by externally supplied carrier cosine and sine samples to produce packed I/Q output. The low word is I and the high word is Q, both signed fixed-point values.

Register interfacesTestbenches: 1

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

radhdl_fft_dma_hpc_top

Top level for the FFT DMA high-performance-controller reference design. Combines FFT datapath, DMA movement, and processing-system interfaces for platform-level validation.

Use Cases

  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.

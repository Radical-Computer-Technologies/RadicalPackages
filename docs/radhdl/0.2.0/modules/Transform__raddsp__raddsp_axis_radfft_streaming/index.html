raddsp_axis_radfft_streaming

Documentation version: 0.2.0 / Subpackage: dsp_transform / Source package: raddsp

Streaming AXI-stream RadFFT architecture. Pushes samples through a streaming transform pipeline for lower buffering pressure and continuous frame processing.

Register interfaces

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

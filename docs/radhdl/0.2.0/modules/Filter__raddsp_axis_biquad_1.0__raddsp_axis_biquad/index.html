raddsp_axis_biquad

Documentation version: 0.2.0 / Package: Filter / Source package: raddsp_axis_biquad_1.0

Multi-channel AXI-stream biquad IIR filter. Applies configurable fixed-point second-order-section coefficients to each channel while preserving ready/valid and frame metadata.

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

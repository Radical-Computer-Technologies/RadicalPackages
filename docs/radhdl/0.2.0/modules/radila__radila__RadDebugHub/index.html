RadDebugHub

Documentation version: 0.2.0 / Package: radila / Source package: radila

AXI-accessible debug hub for RadHDL instrumentation cores. Provides register access and coordination points for debug blocks that need host-visible control, status, and capture data.

Use Cases

  • Debug and observability hooks for generated FPGA systems.

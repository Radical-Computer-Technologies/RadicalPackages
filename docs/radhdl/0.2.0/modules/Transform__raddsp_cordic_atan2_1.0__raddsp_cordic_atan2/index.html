raddsp_cordic_atan2

Documentation version: 0.2.0 / Package: Transform / Source package: raddsp_cordic_atan2_1.0

AXI-stream wrapper around the fixed-point CORDIC atan2 engine. Presents phase-estimation as a ready/valid streaming block for RadDSP dataflow pipelines.

Use Cases

  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.

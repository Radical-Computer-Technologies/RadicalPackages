raddsp_audio_poly_synth

Documentation version: 0.2.0 / Subpackage: dsp_transform / Source package: raddsp

Time-division polyphonic audio synth for register-controlled board designs. A host writes per-voice configuration through cfg_* while explicit Gowin multipliers are reused for wavetable interpolation, waveform-volume, and envelope scaling. Each waveform page stores a positive half period; the opposite half is reconstructed by sign inversion. Voice control bit 20 enables the per-voice one-pole filter; bits [17:2] carry its unsigned Q1.15 coefficient. Bits 1 and 0 remain voice enable/gate.

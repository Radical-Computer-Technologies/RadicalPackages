raddsp_axis_fingerprint_matcher

Documentation version: 0.2.0 / Subpackage: dsp_detection / Source package: raddsp

AXI-stream fingerprint matcher for spectral feature vectors. Compares incoming fingerprint data against reference features and emits match scores for recognition pipelines.

Register interfacesRegister maps: raddsp_axis_fingerprint_matcher_inferred (15 registers)Testbenches: 2

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

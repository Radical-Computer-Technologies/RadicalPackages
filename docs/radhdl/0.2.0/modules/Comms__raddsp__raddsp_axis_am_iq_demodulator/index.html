raddsp_axis_am_iq_demodulator

Documentation version: 0.2.0 / Subpackage: dsp_comms / Source package: raddsp

Register-controlled AXI-stream AM IQ demodulator. Accepts packed signed I/Q samples, optionally rotates them by an internal phase-accumulator DDS carrier for carrier-tuned demodulation, and emits the recovered envelope magnitude. Register control covers enable, internal mixer selection, bypass, phase reset, phase increment, and phase offset.

Register interfacesRegister maps: raddsp_axis_am_iq_demodulator (6 registers)Testbenches: 1

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

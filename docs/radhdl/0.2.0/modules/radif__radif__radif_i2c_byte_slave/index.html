radif_i2c_byte_slave

Documentation version: 0.2.0 / Subpackage: radif_i2c / Source package: radif

I2C byte-register slave for simple board-control buses. Implements the common Linux SMBus-style transaction used by the FPiGA userspace library: one address byte selects an 8-bit register, writes append byte data with auto-increment, and reads return bytes with auto-increment.

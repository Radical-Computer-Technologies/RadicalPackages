radif_axi4_axis_dma

Documentation version: 0.2.0 / Package: radif / Source package: radif

AXI4 memory-mapped to AXI-stream DMA bridge for RadIF systems. Moves register-controlled memory bursts into and out of streaming datapaths with simple status and flow-control signals.

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Memory-mapped control/status integration with software-visible register maps.

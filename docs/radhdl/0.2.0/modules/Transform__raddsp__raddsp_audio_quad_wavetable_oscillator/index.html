raddsp_audio_quad_wavetable_oscillator

Documentation version: 0.2.0 / Subpackage: dsp_transform / Source package: raddsp

Four phase-accumulator oscillators sharing one loadable 16-bit wavetable and one explicit Gowin multiplier for interpolation and per-oscillator gain. Each waveform page stores a positive half period; the second half is reconstructed by sign inversion and adjacent samples are linearly interpolated.

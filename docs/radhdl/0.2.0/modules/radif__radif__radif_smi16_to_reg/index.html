radif_smi16_to_reg

Documentation version: 0.2.0 / Package: radif / Source package: radif

16-bit SMI-style parallel management interface bridge to RadIF registers. Converts external management bus strobes into synchronous register transactions for FPGA peripherals.

Use Cases

  • Memory-mapped control/status integration with software-visible register maps.

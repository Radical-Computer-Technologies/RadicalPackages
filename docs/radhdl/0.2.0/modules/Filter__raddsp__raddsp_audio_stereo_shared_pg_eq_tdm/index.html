raddsp_audio_stereo_shared_pg_eq_tdm

Documentation version: 0.2.0 / Subpackage: dsp_filter / Source package: raddsp

Shared stereo pan, global EQ, and output gain pipeline. This experimental board-level core replaces the standalone pan, EQ, and gain stages with one deterministic TDM schedule using two 27x32 product engines. Each product engine maps to two Gowin MULTALU27X18 blocks.

raddsp_axis_radfft

Documentation version: 0.2.0 / Package: Transform / Source package: raddsp

Top-level AXI-stream RadFFT block. Routes streaming frames through the configured RadDSP FFT implementation and returns ordered frequency-domain samples.

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

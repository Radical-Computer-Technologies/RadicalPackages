radif_reg_bank

Documentation version: 0.2.0 / Package: radif / Source package: radif

Configurable RadIF register bank with software-visible storage. Provides ready/valid read and write responses for a parameterized set of memory-mapped control/status registers.

Use Cases

  • Memory-mapped control/status integration with software-visible register maps.

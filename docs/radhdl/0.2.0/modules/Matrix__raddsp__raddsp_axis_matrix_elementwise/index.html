raddsp_axis_matrix_elementwise

Documentation version: 0.2.0 / Package: Matrix / Source package: raddsp

AXI-stream elementwise matrix arithmetic block. Applies lane-wise arithmetic across matrix-shaped streams for preprocessing and transform pipelines.

Use Cases

  • AXI-Stream datapath integration where module boundaries need explicit ready/valid behavior.
  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.
  • Memory-mapped control/status integration with software-visible register maps.

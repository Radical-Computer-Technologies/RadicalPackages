raddsp_zc_chirp_frame_detector

Documentation version: 0.2.0 / Package: Detection / Source package: raddsp

AXI-stream wrapper around the Zadoff-Chu chirp frame detector. Connects streaming sample input to frame-detection outputs for packet synchronization pipelines.

Use Cases

  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.

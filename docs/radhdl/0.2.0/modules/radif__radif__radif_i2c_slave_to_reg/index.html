radif_i2c_slave_to_reg

Documentation version: 0.2.0 / Subpackage: radif_i2c / Source package: radif

I2C slave bridge to the RadIF register transaction interface. Decodes host I2C register accesses and presents them as synchronous register read/write requests.

Register interfacesTestbenches: 1

Use Cases

  • Memory-mapped control/status integration with software-visible register maps.

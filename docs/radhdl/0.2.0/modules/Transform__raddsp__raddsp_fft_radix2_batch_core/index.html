raddsp_fft_radix2_batch_core

Documentation version: 0.2.0 / Package: Transform / Source package: raddsp

AXI-stream wrapper around the radix-2 batch FFT core. Adapts frame-oriented FFT processing to RadDSP streaming conventions and Vivado IP packaging.

Use Cases

  • Reusable FPGA DSP pipelines that need fixed-point, timing-aware implementation.

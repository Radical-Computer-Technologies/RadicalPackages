radif_reg_interconnect

Documentation version: 0.2.0 / Package: radif / Source package: radif

RadIF register interconnect for routing one host interface to multiple register banks. Decodes address ranges, forwards transactions, and returns the selected peripheral response.

Use Cases

  • Memory-mapped control/status integration with software-visible register maps.

Detailed Description

Portable embedded kernel service ABI for RADLib.

RADEmbeddedKernel exposes a small POSIX-like service contract for bare-metal and simulator backends. The C ABI is the stable boundary; the C++ wrappers are convenience helpers for RADLib applications and examples.
